Pattern Selection Algorithm
Current voice: S (Soprano)
Voice Configuration:
  • S (Soprano): Random, normal time
  • A (Alto): Random, time descent (×0.5)
  • T (Tenor): Phonetic, normal time
  • B (Bass): Phonetic, time descent (×0.5)
Random: Completely random selection. Maximum variety.
Phonetic: Analyzes each syllable:
  • Long consonants → long notes
  • Long vowels/diphthongs → long notes
  • Simple syllables → short notes
⏱️ Time descent: Durations are halved (quarter → eighth, etc.)
New Option: Generate Melody
  • Calculates melodic movement according to consonants.
  • Post-vowel consonants: Raises pitch (e.g. "trans" ↑).
  • Pre-vowel consonants: Lowers pitch (e.g. "bla" ↓).
  • Diphthongs: Amplify the movement.
Automatic: Algorithm changes automatically according to active voice.
