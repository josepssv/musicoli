# 🚀 Musicoli v1.0.0 Release Notes

We are excited to announce the first official release of **Musicoli**, a powerful and intuitive web-based musical composition tool.

Musicoli is designed to make music creation accessible, visual, and rhythmic, focusing on a measure-based workflow and unique text-to-rhythm conversion.

## 🌟 Key Features

*   **Polyphonic Score Editor**: Full 4-voice support (Soprano, Alto, Tenor, Bass) for complex musical arrangements.
*   **Humming System (Tarareo)**: A unique feature that converts text into natural musical rhythms based on syllabification.
*   **Intuitive Color System**: Visualize pitches and tonal functions through a sophisticated color-coding system.
*   **Multiple Editing Modes**: Specialized views for Rhythm, Tonality, Lyrics, and Dynamics.
*   **MIDI Support**: High-quality playback using Soundfonts and instant `.mid` file export.
*   **Pure Web Power**: Built with standard HTML5, CSS, and JavaScript. No installation or build steps required—just open `index.html` and start composing.

## 🛠️ What's New in v1.0
*   Finalized core music engine and rhythmic synchronization.
*   Polished UI with modern aesthetics and responsive layout.
*   Stable MIDI export integration.
*   Complete documentation for the Humming (Tarareo) and Image-to-Melody systems.

## 📥 Getting Started
1.  Download the source code (Zip or Clone).
2.  Open `index.html` in your favorite web browser.
3.  Enjoy composing!
